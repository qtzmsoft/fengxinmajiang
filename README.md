# 房卡麻将（roomcard mahjong）
------
这个程序是参照最近比较火的房卡麻将制作的，所有图片和声音文件来源于网络。如有侵犯您的版权请联系我删除。
### 开发环境
> * macOS Sierra10.12.4
> * COCOS CREATOR1.4.2（下载地址：http://cocos2d-x.org/filedown/CocosCreator_v1.4.2_win）
### 使用说明
> 1. 使用COCOS CREATOR1.4.2创建空工程。
> 2. 把代码覆盖assets目录。
> 3. 使用模拟器运行。
